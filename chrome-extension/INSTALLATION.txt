⚡ QUICK INSTALLATION STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  EXTRACT THIS ZIP FILE
    📁 Unzip to a permanent location (don't delete this folder!)

2️⃣  OPEN CHROME EXTENSIONS
    🌐 Navigate to: chrome://extensions/

3️⃣  ENABLE DEVELOPER MODE
    ⚙️  Toggle "Developer mode" switch (top-right corner)

4️⃣  LOAD UNPACKED EXTENSION
    📂 Click "Load unpacked"
    📁 Select the "chrome-extension" folder
    ✅ Done!

5️⃣  PIN THE EXTENSION (OPTIONAL)
    🧩 Click puzzle icon in Chrome toolbar
    📌 Pin JokerVision extension

🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Extension not showing?
   → Make sure Developer mode is ON
   → Select the folder, not files inside it
   → Refresh the extensions page

❌ Errors appearing?
   → Check browser console (F12)
   → Verify you're on a vehicle listing page
   → Make sure JokerVision app is running

❌ Leads not importing?
   → Check your backend connection
   → Verify the extension is enabled (toggle is blue)
